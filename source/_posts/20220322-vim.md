搞服务器被vim整的欲仙欲死。记录一下。

1. vim

```bash
vim [xxxx]
ctrl + w
:sp [filepath] // 垂直
:vsp [filepath] // 水平
v // 高亮
y // 复制
p // 黏贴
u // 撤销
```

2. linux

```bash
rm -f [filename]
```

3. nginx

```bash
systemctl start nginx
nginx -s reload
nginx -s stop
ps aux | grep nginx
```
